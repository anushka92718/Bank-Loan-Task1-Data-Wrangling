# Data Dictionary – Bank Loan Report

| Column Name     | Data Type | Description                          | Business Meaning |
|----------------|----------|--------------------------------------|------------------|
| id             | Integer  | Unique loan identifier               | Primary key |
| loan_amount    | Numeric  | Loan amount issued                   | Total funding |
| total_payment  | Integer  | Total payment received               | Revenue tracking |
| int_rate       | Float    | Interest rate                        | Profitability |
| dti            | Float    | Debt-to-income ratio                 | Risk indicator |
| issue_date     | Date     | Loan issued date                     | Time analysis |
| loan_status    | String   | Loan status                          | Good/Bad loans |
